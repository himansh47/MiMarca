Professional Skills

HTML5

Hypertext Markup Language, lenguaje de marcas de hipertexto, o HTML es un estándar a cargo del World Wide Web Consortium (W3C) o Consorcio WWW HTML es uasado para definir el contenido de una pagina web, como texto, imágenes, videos, entre otros. HTML busca ser un lenguaje que permita que cualquier pagina web , pueda ser interpretada de la misma forma por cualquier navegador web.

Boostrap 4

Bootstrap es una biblioteca multiplataforma o conjunto de herramientas de código abierto para diseño de sitios y aplicaciones web. Contiene plantillas de diseño con tipografía, formularios, botones, cuadros, menús de navegación y otros elementos de diseño basado en HTML y CSS, así como extensiones de JavaScript adicionales. A diferencia de muchos frameworks web, solo se ocupa del desarrollo front-end.

CSS3

Casacading Style Sheet, hoja de estilo en cascada o CSS, es un lenguaje usado para definir y crear la presentación de un documento estructurado escrito en HTML.

NodeJS

NodeJS Basado en el lenguaje de programación ECMAScript al contrario que la mayoría del código JavaScript, no se ejecuta en un navegador, sino en el servidor.

JQuery

jQuery es una biblioteca multiplataforma de JavaScript, que permite simplificar la manera de interactuar con los documentos HTML, manipular el árbol DOM, manejar eventos, desarrollar animaciones y agregar interacción con la técnica AJAX a páginas web.

JavaScript

JavaScript o JS es un lenguaje de programación interpretado, Es lu lenguaje orientado a objetos,​ basado en prototipos, imperativo, débilmente tipado y dinámico. Se utiliza generalmente en el desarrollo front end para generar interactividad con el usuario y paginas web dinámicas.

Angular 6

Angular es un framework de desarrollo para JavaScript creado por Google. La finalidad de Angular es facilitarnos el desarrollo de aplicaciones web SPA(Single Page Aplication) y además darnos herramientas para trabajar con los elementos de una web de una manera más sencilla y optima. Otro propósito que tiene Angular es la separación completa entre el front-end y el back-end en una aplicación web.
